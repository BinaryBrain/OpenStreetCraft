OpenStreetCraft
===============

OpenStreetMap to Minecraft Map

# Specs

## JSON2Minecraft

```javascript
{
  'elevation': [[z, z, z, ...], [z, z, z, ...], ...],
  'mods': [
    {
      'type': 'stone',
      'coordinates': [x, y, z]
    },
    {
      'type': <something>,
      'coordinates': [x, y, z]
    },
    ...
  ]
}

```
