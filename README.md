OpenStreetCraft
===============

OpenStreetMap to Minecraft Map

# Specs

## JSON2Minecraft

```javascript
{
  'elevation': [[z, z, z, ...], [z, z, z, ...], ...],
  'mods': [
    {
      't': '<data value, ex: 4 = cobblestone>',
      'c': [x, y, z]
    },
    {
      't': <something>,
      'c': [x, y, z]
    },
    ...
  ]
}

```
